# react-responsive-website
React responsive web project

## How to Run

- <h3>Using yarn</h3>

  ```
  $ git clone https://github.com/worjs320/react-responsive-website.git
  $ cd react-responsive-website
  $ yarn install
  $ yarn start
  ```

- <h3>Using npm</h3>

  ```
  $ git clone https://github.com/worjs320/react-responsive-website.git
  $ cd react-responsive-website
  $ npm install
  $ npm start
  ```
